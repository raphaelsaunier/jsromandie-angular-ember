# Angular.js & Ember.js

## Dependencies

ensure you have 

| node  |  v0.10.20 |
| npm   |   1.3.11  |
| bower |   1.2.7   |

## Launch

Launch the slideshow using ``grunt server``

## Create a new slide

`yo reveal:slide "Slide title" --markdown`


## Compiler highlight.js
```sh
python bower_components/highlight.js/tools/build.py -tbrowser javascript html css json xml handlebars
```
